Session Create

Description

Used to create a new session (login the user). Returns a new SessionId for the user. The SessionId is returned in the Session node of the response.

Authentication

Not required.

Authorization

Not required.

Parameters

Name   Description
Username Required An authentication username which references a unique User.
Password Required A user’s login password (the password used to login to the web portal).

Result Element Example

<Result>
  <SessionCreate/>
</Result>

Result Element Schema

Main module: SessionCreate.rng

<?xml version="1.0" encoding="UTF-8"?>
<grammar xmlns="http://relaxng.org/ns/structure/1.0">
  <!--
    
    SessionCreate Result
    
  -->
  <define name="Result.content" combine="choice">
    <ref name="SessionCreate"/>
  </define>
  <!--
    
    SessionCreate Element
    
  -->
  <define name="SessionCreate">
    <element name="SessionCreate">
      <zeroOrMore>
        <ref name="AnyAttribute"/>
      </zeroOrMore>
      <ref name="SessionCreate.content"/>
    </element>
  </define>
  <define name="SessionCreate.content">
    <zeroOrMore>
      <ref name="SessionCreate.content.open"/>
    </zeroOrMore>
  </define>
  <define name="SessionCreate.content.open">
    <element>
      <anyName/>
      <ref name="Anything"/>
    </element>
  </define>
</grammar>

Session Destroy

Description

Used to destroy an existing session (logout the user).

Authentication

Not required.

Authorization

This action requires an Authenticated Session.

Parameters

None.

<Result>
  <SessionDestroy/>
</Result>
